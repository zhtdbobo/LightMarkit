- [x] **M1.2** 集成 CodeMirror 6 编辑器内核
  - ✅ 安装 CodeMirror 6 核心包与 Markdown 语言包
  - ✅ 创建 Editor 组件，配置基础编辑器
  - ✅ 实现 Markdown 语法高亮
  - ✅ 编写完整测试用例（14 个测试，全部通过）
  - ✅ 验证：能输入文本，语法正确着色

- [ ] **M1.3** 实现 markdown-it 渲染管线
  - 安装 markdown-it 与 GFM 插件（表格/任务列表/删除线）
  - 创建 Preview 组件，实现 Markdown → HTML 转换
  - 配置渲染样式（Typography CSS）
  - 验证：预览区正确渲染表格、任务列表、删除线

- [ ] **M1.4** 实现分屏布局与模式切换
  - 实现三栏布局（侧栏/编辑区/预览区）
  - 实现三种模式：纯编辑 / 分屏 / 纯预览
  - 实现 `Ctrl+/` 快捷键切换模式
  - 实现可拖拽分割条
  - 验证：模式切换流畅，分割条拖动正常

- [ ] **M1.5** 实现文件读写（Rust 端）
  - 在 Rust 端实现 `file_read` / `file_write` Tauri command
  - 实现原子写入机制（临时文件 + rename）
  - 前端封装 IPC 调用的类型安全 API
  - 验证：能打开/保存单个 .md 文件，崩溃不损坏文件

- [ ] **M1.6** 实现自动保存

  - 状态栏显示"已自动保存"提示
  - 验证：停止输入 500ms 后自动保存，强制退出不丢数据

- [ ] **M1.7** 实现编辑辅助功能
  - 实现括号/星号自动配对

  - 实现选中文字粘贴 URL 自动生成链接
