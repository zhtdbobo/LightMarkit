## M2 — 导入导出功能

**目标**：支持所有渲染内容的完整导出（HTML、PDF、Markdown）

- [x] **M2.1** 实现文件夹打开与多文件管理
  - ✅ 实现 `scan_folder` Tauri command 扫描文件夹中的 Markdown 文件
  - ✅ 支持递归扫描子目录
  - ✅ 创建 FileList 组件显示文件树
  - ✅ 实现文件夹打开 UI 和快捷键（Ctrl+Shift+O）
  - ✅ 支持从文件列表选择并打开文件
  - ✅ 编写完整测试用例（folderApi 测试 4 个，FileList 测试 8 个）

- [x] **M2.2** 实现 HTML 导出功能
  - ✅ 实现 `export_html` Tauri command
  - ✅ 完整渲染 Markdown 语法（包括链接、图片、表格、任务列表等）
  - ✅ 导出的 HTML 包含完整样式（与预览区样式一致）
  - ✅ 支持 Mermaid 图表渲染（通过 CDN）
  - ✅ 自动提取文档标题
  - ✅ 编写完整测试用例（exportApi 测试 4 个）

- [x] **M2.3** 集成 Mermaid 图表支持
  - ✅ 安装 mermaid 依赖
  - ✅ 在预览区渲染 Mermaid 图表
  - ✅ 扩展 markdown-it 渲染器支持 Mermaid 代码块
  - ✅ 添加 Mermaid 样式
  - ✅ 导出时正确渲染 Mermaid 图表

- [x] **M2.4** 实现 PDF 导出功能
  - ✅ 添加 headless_chrome 依赖
  - ✅ 实现 `export_pdf` Tauri command
  - ✅ 使用无头 Chrome 渲染 HTML 为 PDF
  - ✅ 导出带完整样式和 Mermaid 图表的 PDF
  - ✅ 支持页面设置（A4 纸张、页边距）
  - ✅ 优化分页（避免表格、图片断裂）
  - ✅ 编写完整测试用例（exportApi 测试 2 个）

- [x] **M2.5** 实现 Markdown 标准化导出
  - ✅ 实现 `export_markdown` Tauri command
  - ✅ 导出格式化的 Markdown 文件
  - ✅ 保留所有渲染元素（表格、任务列表、Mermaid 等）
  - ✅ 清理多余空行和行尾空格
  - ✅ 编写完整测试用例（exportApi 测试 3 个）

- [ ] **M2.6** 实现文件夹批量导出功能
  - 支持将整个文件夹导出为 HTML/PDF
  - 保持文件夹结构
  - 生成索引页面
  - 支持链接转换（相对路径处理）
  - 编写测试用例
