# LightMarkit 导出功能规范

> 本文档定义所有导出功能的规范和实现要求

## 导出格式支持

LightMarkit 支持将 Markdown 文档导出为以下格式：

1. **HTML** - 带完整样式的网页文档
2. **PDF** - 适合打印和分享的 PDF 文档
3. **Markdown** - 标准化格式的 Markdown 文件

## 导出内容要求

所有导出格式必须包含**预览区能够渲染的所有内容**：

### 基础 Markdown 语法
- ✅ 标题（H1-H6）
- ✅ 段落和换行
- ✅ 粗体、斜体、删除线
- ✅ 代码块和行内代码
- ✅ 引用块
- ✅ 有序列表和无序列表
- ✅ 任务列表
- ✅ 链接
- ✅ 图片
- ✅ 分隔线

### GFM 扩展语法
- ✅ 表格
- ✅ 删除线
- ✅ 任务列表（带复选框）
- ✅ 自动链接

### 高级功能
- ✅ Mermaid 图表
- 🔄 数学公式（KaTeX）- 待实现
- 🔄 代码高亮（Shiki）- 待实现

## HTML 导出规范

### 文档结构
```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>{文档标题}</title>
    <style>{完整 CSS 样式}</style>
    <script type="module">{Mermaid 初始化}</script>
</head>
<body>
    {渲染后的 HTML 内容}
</body>
</html>
```

### 样式要求
- 使用与预览区完全相同的 CSS 样式
- 包含完整的排版样式（字体、颜色、间距）
- 包含所有特殊元素样式（表格、代码块、引用块等）
- 包含 Mermaid 图表样式
- 支持打印样式（@media print）

### Mermaid 集成
- 通过 CDN 引入 Mermaid 库（https://cdn.jsdelivr.net/npm/mermaid@10/dist/mermaid.esm.min.mjs）
- 在文档加载时自动初始化和渲染所有图表
- 确保离线打开时也能正常显示（考虑嵌入式选项）

## PDF 导出规范

### 渲染方式
- 使用 HTML 作为中间格式
- 通过 headless Chrome 或类似工具渲染为 PDF
- 可选方案：wkhtmltopdf、puppeteer

### 页面设置
- **默认纸张**：A4（210mm × 297mm）
- **默认页边距**：上下 20mm，左右 25mm
- **页眉页脚**：可选（文档标题、页码）
- **颜色模式**：彩色输出

### 内容要求
- 所有 Markdown 元素完整渲染
- Mermaid 图表转换为静态图片
- 代码块保持格式和语法高亮
- 表格正确分页（避免断裂）
- 链接保留（可点击）

### 质量要求
- 分辨率：至少 150 DPI
- 字体嵌入：确保所有字体正确显示
- 图片质量：保持原始分辨率

## Markdown 导出规范

### 格式标准化
- 使用标准 CommonMark + GFM 语法
- 统一的换行和空行规则
- 标准化的列表缩进（2 空格）
- 规范的代码块标记（```language）

### 特殊元素处理
- **Mermaid 图表**：保留 ```mermaid 代码块
- **任务列表**：使用 `- [ ]` 和 `- [x]` 格式
- **表格**：使用 GFM 表格语法
- **图片**：保留 `![alt](url)` 格式

### 路径处理
- 相对路径：保持相对路径不变
- 绝对路径：转换为相对路径（如果可能）
- 外部 URL：保持不变

### 元数据处理
- 可选添加 Front Matter（YAML）
- 包含标题、创建时间、修改时间等

## 批量导出功能

### 文件夹导出
- 支持将整个文件夹导出为 HTML/PDF
- 保持原始文件夹结构
- 生成索引页面（index.html）
- 处理文档间的相对链接

### 链接转换
- 内部链接：转换为相对路径
- 跨文档链接：保持有效性
- 外部链接：保持不变

### 索引页面
- 列出所有文档的层级结构
- 提供可点击的导航链接
- 显示文档标题和简介（可选）

## API 设计

### Rust 端 Commands

```rust
// 导出 HTML
#[tauri::command]
async fn export_html(file_path: String, html_content: String, title: String) -> Result<(), String>

// 导出 PDF
#[tauri::command]
async fn export_pdf(file_path: String, html_content: String, title: String) -> Result<(), String>

// 批量导出文件夹
#[tauri::command]
async fn export_folder(folder_path: String, output_path: String, format: String) -> Result<(), String>
```

### 前端 API

```typescript
// 导出单个文件
export async function exportHtml(filePath: string, htmlContent: string, title: string): Promise<void>
export async function exportPdf(filePath: string, htmlContent: string, title: string): Promise<void>

// 批量导出
export async function exportFolder(folderPath: string, outputPath: string, format: 'html' | 'pdf'): Promise<void>
```

- Linux（主流发行版）

- 导出 PDF 可在标准 PDF 阅读器中打开
