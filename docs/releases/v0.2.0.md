## 版本说明

LightMarkit v0.2.0 增加了面向中文 Markdown 写作的斜杠快捷命令，并改进了编辑器标题和表格的显示一致性。

## 新增功能

- 在编辑器行首输入 `/` 打开中文快捷命令菜单。
- 提供正文、一级至三级标题、无序列表、有序列表、待办事项、引用、代码块、分隔线、链接、图片和表格等 13 个常用命令。
- 支持输入中文筛选命令，并使用方向键、Enter 或鼠标完成选择。

## 体验改进

- 快捷命令菜单默认显示 10 项，仅显示中文名称。
- 菜单支持明暗主题，并在窄窗口下保持可用。
- 表格快捷命令生成左对齐列，表头与内容默认保持一致。

## 问题修复

- 修复编辑模式下 Markdown 标题出现下划线的问题。
- 避免在普通文本和 URL 中误触发快捷命令菜单。
