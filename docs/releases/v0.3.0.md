## 版本说明

LightMarkit v0.3.0 增加了文档大纲、工作区状态恢复和外部文件变更监听，并重做了编辑区与预览区的滚动定位体验。

## 新增功能

- 新增可调整宽度的文档大纲，展示 Markdown 标题层级。
- 支持当前章节高亮，并可点击大纲标题跳转到对应位置。
- 支持恢复文件夹、当前文件、大纲显隐和侧栏宽度。
- 支持监听当前文件的外部变更，在没有本地未保存修改时自动刷新。

## 体验改进

- 编辑区与预览区使用 Markdown 源码块锚点进行滚动同步。
- 大纲按视口阅读探针识别当前段落，不再严格跟随视口顶端行。
- 表格与 Mermaid 图表优先按块上边界识别，减少长内容滚动时的高亮跳动。
- 优化预览区顶部留白和渲染复用，降低内容刷新时的视觉位移。

## 问题修复

- 避免分屏同步滚动反向覆盖实际滚动来源的大纲高亮。
- 避免外部文件更新覆盖尚未保存的本地编辑内容。
